# 《霓虹清道夫》Story Bible v1

## 一句话概念

近未来旧金山，城市污染事故被伪装成普通交通灾难。玩家观看一部 AI 电影短片，并在关键动作片段短暂接管主角，凭操作数值进入不同结局影片。

## 类型和气质

- 类型：AI 电影互动游戏、赛博 noir、动作惊悚、怪异清道夫题材。
- 画面目标：顶级电影感、夜雨城市、霓虹反射、湿地、车灯、雾、强轮廓光。
- 情绪目标：爽感、紧张、遗憾、代价感。
- 体验原则：玩家不是频繁点选项，而是在最想插手的时候真正插手。

## 世界观

2037 年，旧金山的地下交通系统被一类“拟态污染”侵入。它会伪装成车祸、火灾、设备故障，吞掉证据并改写摄像头画面。城市雇佣一群半官方清道夫处理这些“事故”，但多数清道夫只负责善后，不追问源头。

主角林夏发现，污染并不是自然灾害，而是一家公司用 AI 视频伪造事故现场，掩盖活体实验。

## 主角

林夏：前特技车手，现为城市清道夫。她擅长驾驶、短兵战斗和现场判断。过去一次营救失败让她对“迟到一步”极度敏感。

阿洛：车载 AI 助手。它能标记路线、计算风险、预测污染扩散，但不能替玩家做决定。

维克多：生物影像公司安保负责人。表面是城市安全承包商，实际负责转运污染核心。

米娅：内部证人。她掌握一段原始视频，可以证明污染事件是人为制造。

## Demo 核心循环

```text
观看电影段 -> 进入接管窗口 -> 玩家操作 -> 计算数值 -> 播放对应电影分支
```

## Demo 分支

```text
A0 开场：雨夜旧金山，米娅被押上黑色车队
  -> I1 追车接管
      -> C1 完美追上：爽快进入港口
      -> C2 受损追上：带伤进入港口
      -> C3 跟丢：迟到进入港口
  -> A2 港口仓库：维克多启动证据伪造上传
  -> I2 Boss 接管
      -> E1 完美胜利：救人，保住证据
      -> E2 代价胜利：救人，但维克多逃走
      -> E3 遗憾结局：救人失败或证据被污染
```

## 禁忌

- 不做普通丧尸片。
- 不照搬参考视频人物和桥段。
- 不让玩家一直打断电影节奏。
- 不依赖实时 AI 生成作为第一版玩法核心。
- 不让分支只是文字变化，必须对应不同影片结果。
